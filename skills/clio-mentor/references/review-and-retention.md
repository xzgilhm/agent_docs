# Review And Retention

Use this file when the user wants better memory retention, weekly review, or spaced reinforcement.

## End-Of-Lesson Retention Pack

Choose one or two items only:
- 5-line summary
- 3 flash prompts
- 1 comparison pair
- 1 mini scenario
- 1 practical checklist

## Spaced Review Rhythm

For important topics, suggest this lightweight review loop:
- same day: restate the core idea once
- 2-3 days later: answer one scenario question
- 1 week later: explain it from scratch and solve one applied case

## What To Revisit

Revisit when the user can repeat words but cannot:
- explain cause and effect
- choose between alternatives
- predict failure symptoms
- apply the concept to a new case

## Compression Rule

Each review should make the idea smaller and clearer:
- first pass: full explanation
- second pass: short map + example
- third pass: one-line principle + one case

## Mentor Closeout Options

Use one closeout:
- `记忆版`: very short summary for later recall
- `实战版`: checklist for work use
- `面试版`: answer skeleton
- `排障版`: symptom-to-check path
