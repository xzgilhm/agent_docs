# AWS 学习路线图

这是当前 AWS 学习路线图，按“认路 -> 能讲 -> 能做 -> 能排障 -> 能迁移”的顺序推进。

## 学习进度

当前进度：**2 / 10**

当前阶段：**IAM**

计划章节：
1. AWS 全局地图：Account / Region / AZ / Global services / core layers
2. IAM：identity、role、policy、temporary credentials
3. VPC：subnet、route、IGW、NAT、SG、NACL
4. Compute：EC2、AMI、EBS、ASG、ELB
5. Storage & Data：S3、EBS、EFS、RDS、DynamoDB、ElastiCache
6. Observability & Ops：CloudWatch、CloudTrail、Config、SSM
7. Security & Governance：KMS、Secrets、Organizations、cost/safety controls
8. End-to-end architecture drills
9. Common failure scenarios
10. Production migration and extension topics

## 建议节奏

每一节按以下顺序：

1. 先看 GitHub 里的章节
2. 回答 1 个短小考题
3. 基本答对后继续下一节
4. 逐步把 AWS 从名词记忆变成系统地图、操作路径和排障框架
